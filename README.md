# Venkata Naga Sai — Portfolio

A premium, modern personal portfolio website built with pure HTML, CSS, and JavaScript. Features a sleek dark theme, glassmorphism effects, smooth animations, and a fully responsive design.

## ✨ Features

- **Dark Premium Theme** — Carefully crafted color palette with purple accent gradients
- **Glassmorphism UI** — Frosted glass card effects with subtle borders
- **Typing Animation** — Dynamic hero text cycling through roles
- **Cursor Glow Effect** — Ambient light that follows cursor movement
- **Floating Particles** — Subtle animated background particles
- **Scroll Reveal** — Elements animate into view on scroll
- **Animated Counters** — Statistics count up when visible
- **Fully Responsive** — Mobile-first design with smooth transitions
- **SEO Optimized** — Semantic HTML5, meta tags, and proper heading hierarchy

## 🛠️ Tech Stack

- HTML5
- CSS3 (Vanilla — no frameworks)
- JavaScript (Vanilla — no dependencies)
- Google Fonts (Inter, JetBrains Mono)

## 🚀 Quick Start

Simply open `index.html` in your browser, or deploy to GitHub Pages.

### Deploy to GitHub Pages

1. Push this repository to GitHub
2. Go to **Settings** → **Pages**
3. Set source to **Deploy from a branch** → **main** / **root**
4. Your portfolio will be live at `https://<your-username>.github.io/<repo-name>/`
